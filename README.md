# Mini Zelda im Browser

Ein kleines selbstgebautes Zelda-inspiriertes Spiel mit verstärktem **3D-Look**, direkt in `index.html` (ohne externe Seiten oder Links).

## Was ist neu?

- Link hat einen klareren, BOTW-inspirierten Look (blaue Tunika, Haare, sichtbares Gesicht).
- Das einsammelbare Schwert wurde auf einen **Masterschwert-Look** umgestellt.
- Nach dem Einsammeln gilt:
  - **F** beim ersten Mal: Masterschwert ziehen.
  - **F** erneut: Schwertschlag ausführen.
  - Solange das Schwert gezogen ist, ist der Bogen auf Links Rücken sichtbar.
  - Solange das Schwert nicht gezogen ist, trägt Link den Bogen in der Hand und das Schwert auf dem Rücken.

## Spielen

1. Datei `index.html` im Browser öffnen.
2. Link bewegen mit **WASD** oder **Pfeiltasten**.
3. Mit **Leertaste** den Bogen nutzen.
4. Nach Schwertfund: **F** zieht das Masterschwert, **F** erneut führt einen Schlag aus.
5. Ziel: **8 Rubine** sammeln und danach den **Tempel (goldenes Feld)** erreichen.
6. Mit **R** neu starten.
